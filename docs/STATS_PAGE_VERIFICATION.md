## 테스트 방법

### 로컬 개발 서버 실행
```bash
pnpm dev
```

### 브라우저에서 확인
1. `http://localhost:3000/stats` 접속
2. 위 체크리스트 항목들을 하나씩 확인

### 에러 테스트 방법
에러 처리를 테스트하려면:
1. 개발자 도구 > Network 탭 열기
2. 네트워크 요청 차단 (Offline 모드 또는 Request Blocking)
3. 페이지 새로고침
4. 에러 메시지와 재시도 버튼 확인
5. 재시도 버튼 클릭하여 정상 동작 확인
