# 홈페이지 통합 테스트 체크리스트

## 1. 필터 조합 테스트

### 1.1 지역 필터
- [ ] 전체 → 서울 선택 시 서울 지역 관광지만 표시
- [ ] 서울 → 부산 변경 시 부산 지역 관광지로 즉시 업데이트
- [ ] 부산 → 전체 선택 시 모든 지역 관광지 표시
- [ ] URL 파라미터 `areaCode` 정확히 반영

### 1.2 관광 타입 필터
- [ ] 관광지(12) 선택 시 관광지 타입만 표시
- [ ] 문화시설(14) 추가 선택 시 관광지 + 문화시설 모두 표시
- [ ] 관광지 선택 해제 시 문화시설만 표시
- [ ] 모든 타입 선택 해제 시 모든 타입 표시
- [ ] URL 파라미터 `contentTypeId` 정확히 반영

### 1.3 반려동물 필터
- [ ] 반려동물 동반 가능 토글 시 필터링 동작
- [ ] 크기별 필터 (소형, 중형, 대형) 동작

### 1.4 정렬 옵션
- [ ] 최신순 정렬 동작 확인
- [ ] 이름순 정렬 동작 확인
- [ ] URL 파라미터 `sort` 정확히 반영

### 1.5 필터 조합
- [ ] 지역 + 관광 타입 조합 동작
- [ ] 지역 + 관광 타입 + 반려동물 조합 동작
- [ ] 모든 필터 동시 적용 시 정확한 필터링
- [ ] 필터 초기화 버튼 동작

## 2. 검색 기능 테스트

### 2.1 기본 검색
- [ ] 키워드 입력 후 엔터로 검색 실행
- [ ] 검색 버튼 클릭으로 검색 실행
- [ ] 검색 결과 개수 정확히 표시
- [ ] 검색 결과 없음 메시지 표시

### 2.2 검색 + 필터 조합
- [ ] 검색어 + 지역 필터 조합
- [ ] 검색어 + 관광 타입 필터 조합
- [ ] 검색어 + 모든 필터 조합
- [ ] 검색어 삭제 시 일반 목록 모드로 전환

### 2.3 검색 결과 표시
- [ ] 검색 결과 헤더에 키워드 및 개수 표시
- [ ] 검색 결과가 지도에 마커로 표시
- [ ] 검색 결과 없음 시 빈 상태 메시지 표시

## 3. 페이지네이션 테스트

### 3.1 기본 동작
- [ ] 페이지 1에서 15개 항목 표시
- [ ] 페이지 2 클릭 시 16-30번 항목 표시
- [ ] 총 개수 및 현재 범위 정확히 표시
- [ ] URL 파라미터 `page` 정확히 반영

### 3.2 버튼 동작
- [ ] 이전 버튼 클릭 시 이전 페이지로 이동
- [ ] 다음 버튼 클릭 시 다음 페이지로 이동
- [ ] 페이지 1에서 이전 버튼 비활성화
- [ ] 마지막 페이지에서 다음 버튼 비활성화

### 3.3 경계 처리
- [ ] 잘못된 페이지 번호 (0, 음수) 처리
- [ ] 존재하지 않는 페이지 번호 (999) 처리
- [ ] URL 직접 입력 시 올바른 페이지 표시

### 3.4 필터/검색과 조합
- [ ] 필터 변경 시 페이지 1로 리셋
- [ ] 검색 실행 시 페이지 1로 리셋
- [ ] 페이지 변경 시 필터 조건 유지

## 4. 지도-리스트 연동 테스트

### 4.1 리스트 → 지도
- [ ] 리스트 항목 클릭 시 지도 이동
- [ ] 리스트 항목 클릭 시 마커 강조
- [ ] 리스트 항목 클릭 시 인포윈도우 표시
- [ ] 모바일에서 리스트 클릭 시 지도 탭으로 전환

### 4.2 지도 → 리스트
- [ ] 마커 클릭 시 인포윈도우 표시
- [ ] 마커 클릭 시 리스트 항목 강조
- [ ] 여러 마커 연속 클릭 시 마지막 선택 항목 유지

### 4.3 선택 상태 관리
- [ ] 페이지 변경 시 선택 상태 초기화
- [ ] 필터 변경 시 선택 상태 초기화
- [ ] 관광지 목록 변경 시 선택 상태 자동 초기화

## 5. URL 파라미터 동기화 테스트

### 5.1 필터 변경
- [ ] 필터 변경 시 URL 즉시 업데이트
- [ ] 브라우저 히스토리에 추가
- [ ] URL 복사 후 새 탭에서 열기 시 동일한 필터 적용

### 5.2 URL 직접 입력
- [ ] `/?areaCode=1` 입력 시 서울 지역 필터 적용

- [ ] 잘못된 URL 파라미터 처리

### 5.3 브라우저 네비게이션
- [ ] 뒤로가기 버튼 동작
- [ ] 앞으로가기 버튼 동작
